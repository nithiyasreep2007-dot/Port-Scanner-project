Port Scanner Project

This is a simple Python-based port scanner using socket programming.

Features
- Scans common ports (21, 22, 80, 443)
- Detects open and closed ports
- Uses Python socket library

echnologies Used
- Python
- Socket Programming

How to Run
1. Install Python
2. Run:
   python port_scanner.py
3. Enter target IP

Output Example
Port 80 is OPEN
Port 22 is CLOSED
